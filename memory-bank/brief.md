# 项目目标

## 核心定位
纯前端天气仪表盘应用，提供实时天气数据可视化展示，部署在 GitHub Pages。

## 功能目标
- 城市搜索：支持中文区县/城市名称搜索
- 地理定位：浏览器定位 + 反向地址解析
- 实时天气：温度、湿度、风速、日出时间
- 天气动画：Lottie 动画 + CSS fallback
- 背景特效：Canvas 粒子动画（阳光、雨、雪、云）
- 5天预报：温度范围 + 降水概率
- 24小时温度图：Chart.js 折线图
- AQI/UVI：空气质量 + 紫外线指数
- 主题切换：日间/夜间模式（自动根据日出日落）
- 城市收藏：localStorage 存储，最多8个
- 自动刷新：每60秒自动更新

## 设计风格
玻璃态（glassmorphism）设计：
- 毛玻璃卡片 + backdrop-filter
- 渐变背景：日间蓝紫粉、夜间深蓝黑
- 大圆角：24px 卡片 + 18px 小组件
- 响应式：适配 420px 宽度移动端

## 用户偏好
- 温暖手绘风格（非冷冰冰 AI 美学）
- 简洁内容，避免信息过载
- 注重 UI/UX 设计细节
